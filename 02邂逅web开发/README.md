# 网站和页面的关系

.什么是网页？
.网页：就是通过浏览器打开的网页，网页就是通过浏览器来显示的，浏览器就是通过浏览器内核来显示的。

用户角度
.用户在浏览器输入一个网站
.浏览器会找到对应的服务器地址，请求静态资源（可以存放在世界任何一个地方）

.输入网站后会找到 DNS 服务器解析域名，找到对应的 IP 地址
.浏览器会通过 IP 地址找到服务器，请求服务器中的静态资源
.浏览器会通过服务器返回的静态资源（html，css，js），渲染成页面（渲染引擎：html，css）（js 引擎：js）
